
                   *  Команды для администрирования  *
* quit                            
Завершает работу бота. Only for admin.

* join room nick                  
Зайти в конференцию room под ником nick. Only for admin.

* lang_update lang                
Перезагружает файл сообщений языка lang. Only for admin.

* invite jid [room]               
Пригласить jid в конференцию room или текущую, если room не указан.
Only for admin.

* msg jid message
Заставляет послать сообщение message пользователю с указанным jid.
Only for admin.

* greet jid room@server greet
Устанавливает приветствие greet при появлении пользователя с указанным jid в
комнате room@server. Only for admin.

* sulci_set variable=value
Устанавливает внутреннюю переменную в новое значение. Only for admin.



                   *  Список переменных  *
* msg_limit
Использовать приват для ответов, размер которых превышает указанный лимит
в байтах.



                   *  Сеть  *
* ping [nick]
пинг до пользователя с указанным ником или до себя, если ник не указан.

* dns ip_addr
Показывает результат преобразования имени хоста в ip адрес.

* dns name
Показывает результат преобразования ip адреса в имя хоста.

* hostip ip_address
Показывает географическое положение ip адреса согласно сведениям
http://www.hostip.info/

* tld tld
Показывает имя страны по названию домена верхнего уровня(напр. ru - это Russia).

* stats server.name
Показывает статистику об общем количестве пользователей, пользователей о онлайне
и пр. информацию для указанного Джаббер сервера.

* uptime server.name
Показывает время, прошедшее с момента последней остановки Джаббер сервера.



                   *  Поиск  *
* google query
Поиск через Google, выдаёт первый результат.

* google_adv start limit query
Поисковый запрос к Google. Возвращаются limit результатов, начиная с start.

* gspell query
Возвращает совет Google по написанию запроса.

* blogs text
Поиск text через http://blogs.yandex.ru/ (поиск по блогам и форумам)



                   *  Перевод  *
* tr list
Выводит список кодов языков, для перевода.

* tr lang text
Переводит указанный текст через http://www.translate.ru/ в соответствии с кодом
языка.

* mueller text
Англо-русский словарь.

* dict -list
Показывает список баз данных с dict.org или другого сервера, указанного
в конфигурации.

* dict [db] word
Запрашивает значение слова word в базе данных db, через dict.org или другой
сервер из конфигурации. При отсутствии db, поиск осуществляется во всех базах
сервера.

* gtr lang lang text
Переводит указанный текст с помощью сервиса перевода Google в соответствии с
кодом языка.



                   *  Статистика  *
* version [nick]
Выводит версию клиента nick'а или, при его отсутствии, отправителя запроса.

* vcard nick
Выводит vCard указанного ника, если есть

* time [nick]
Выводит локальное время на компьютере запросившего или nick'а, если он указан.


* status [nick]
Выводит статус указанный в клиенте запросившего или, при указании
nick'а - статус в клиенте nick'а.

* idle [nick]
Показывает время бездействия nick'а или отправителя запроса, если nick'не указан.

* version_server server.name
Выводит версию серверного программного обеспечения на указанном Джаббер сервере.

* seen nick
Показывает когда последний раз nick появлялся и его последнюю фразу.

* talkers [nick]
Без указания nick'а показывает TOP 10 говорунов, иначе стастическую информацию
о болтливости указанного nick'а. nick может представлять из себя SQL'образное
выражение с символами [%_].

* !!!count
Возвращает число пар слов в болтологической базе текущей конференции.

* !!!top
OP 10 пар слов по этой же базе.

* wtf word
Возвращает последнее определение термина word.

* dfn word=[text]
Задает определение термина word, как text. Если text отсутствует - удаляет
определение для word.

* wtfcount [word]
Возвращает кол-во определений всего или для термина word, если указано.


* wtflall word
Возвращает список всех определений термина word.

* wtfrand [word]
Возвращает случайное определение из всех существующих или из определений термина
word, если указано.



                   *  Разное  *
* wz code
Выводит прогноз погоды с http://weather.noaa.gov/ для города, обозначенного 4-х
буквенным кодом code.


* curr 10 usd rur
Преобразователь денежных единиц.

* curr list
Показывает список валют.

* rpn 1 3 + ...
Калькулятор в польской нотации.

* calc 1 + 3
Калькулятор в прямой нотации.

* tryme
"Русская рулетка".
